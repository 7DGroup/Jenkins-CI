Ein gütiges Java-String-Format fü den Header eines Changelog-Eintrags. Dieser unterstützt die folgenden vier Platzhalter im String-Format:

  1. Autor des Commits
  2. ID oder Revision des Commits
  3. Beschreibung des Commits
  4. Datum des Commits im Format, das im Feld Datumsformat angegeben wurde.
Nach diesen Informationen folgt eine Liste der bearbeiteten Pfade, wie im Feld Format eines Dateieintrags angegeben.
